# Conference Project Scheduling Analysis

This project evaluates schedule feasibility, time-cost trade-offs, crashing scenarios, and delivery risk for a hypothetical analytics innovation conference.

## Business Problem

The project must be delivered inside a fixed deadline while maintaining an acceptable probability of on-time completion. The analysis compares baseline scheduling against selective crashing scenarios to find a plan that improves schedule confidence without unnecessary cost escalation.

## What This Project Demonstrates

- Critical Path Method (CPM)
- PERT-based uncertainty analysis
- Schedule crashing and time-cost trade-off analysis
- Scenario comparison across baseline and accelerated plans
- Earned Value Management (EVM) tracking logic
- ProjectLibre schedule files for baseline and crashed scenarios
- Risk-aware recommendation for project delivery

## Tools Used

- ProjectLibre
- Microsoft Excel
- Critical path analysis
- PERT
- EVM
- Scenario and sensitivity analysis
